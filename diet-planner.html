मंगळवारपराठे + दहीचिकन + भातकेळी शेकपनीर + रोटी
बुधवारअंडी + ब्रेडराजमा भातड्रायफ्रूटडाळ + चपाती
गुरुवारपोहे + दूधमासे + भातपनीरएग भुर्जी
शुक्रवारबेसन चिलाछोले + भातफळे + दहीचिकन + रोटी
शनिवारउपमा + दूधबिर्याणीशेंगदाणेखिचडी
रविवारमिसळ पावमटण + भातप्रोटीन शेकदही भात

📅 साप्ताहिक वेळापत्रक

दिवसनाश्तादुपारस्नॅकरात्री
सोमवारओट्स + फळेडाळ + भात (कमी)ग्रीन टीसूप + सॅलड
मंगळवारउकडलेली अंडीग्रिल्ड चिकनसफरचंदचपाती भाजी
बुधवारमूग चिलाखिचडीबदामदलिया
गुरुवारफ्रूट बाउलग्रिल्ड मासेताकमूग सूप
शुक्रवारस्प्राउट्सभाज्या सूप + रोटीकाकडीअंडा भुर्जी
शनिवारओट्सब्राउन राइस डाळमखानेसॅलड
रविवारलिंबू पाणी + उपमारसम भातफळेहळद दूध

💡 महत्त्वाच्या टिप्स

  • ✅ दररोज किमान 8-10 ग्लास पाणी प्या

  • ✅ रात्रीचे जेवण 8 वाजेपूर्वी करा
  • ✅ जंक फूड, कोल्ड्रिंक्स पूर्णपणे टाळा
  • ✅ हळूहळू चावून खा, घाई करू नका

  • ⚠️ कोणताही आहार बदल करण्यापूर्वी डॉक्टरांचा सल्ला घ्या
